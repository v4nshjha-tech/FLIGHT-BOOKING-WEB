# 📖 About The Project

 Flight Booking Web System is a complete web-based airline reservation platform that enables administrators to efficiently manage flights, bookings, agencies, routes, and passengers while providing users with a seamless flight search and booking experience.

The project follows a modular architecture and integrates database-driven operations using PHP and MySQL for reliable data management.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/v4nshjha-tech/FLIGHT-BOOKING-WEB.git
```

Move the project into:

```text
xampp/htdocs/
```

Import the SQL database from the **sql/** folder using **phpMyAdmin**.

Start:

- ✅ Apache
- ✅ MySQL

Open in browser:

```text
http://localhost/FLIGHT-BOOKING-WEB/
```
